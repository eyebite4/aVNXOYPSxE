# Python南京某高校校园外卖点餐系统简介

## 核心功能
- 登录注册
- 美食信息展示
- 美食详情查看
- 下单功能
- 添加到购物车
- 美食分类浏览
- 个人中心管理
- 美食信息管理
- 订单信息管理
- 用户管理
- 安全检查管理
- 商家管理
- 工作人员管理

## 适用场景
- 高校校园内
- 提供便捷的外卖点餐服务
- 适用于学生、商家、工作人员和管理人员

## 技术特点
- 使用 Django 框架构建后端
- 基于 MySQL 数据库进行数据存储
- 前端采用 Vue.js 技术
- 支持远程调试，保证系统稳定运行
